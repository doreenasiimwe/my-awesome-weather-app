# Weather Search App

## Description

This weather application allows users to search for a city and view real-time weather information.

When a user searches for a city (for example, Sydney), the application displays:

* City name
* Current temperature
* Current weather conditions
* Current day and time
* Humidity
* Wind speed

## Features

* Search weather by city name
* Display real-time weather data
* Show current temperature and weather conditions
* Display the current day and time
* Show humidity and wind speed
* Clean and responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript
* Axios
* Shecodes Weather API

## How to Use

1. Enter a city name in the search field.
2. Click the Search button.
3. View the current weather information for the selected city.
